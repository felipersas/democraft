Esta é a menor demo útil do Democraft: uma cena, um goto, um establish e uma legenda. Ela compila, captura e renderiza de ponta a ponta.

## O arquivo de demo

```ts title="demo.ts"
import { byTestId, defineDemo } from "@democraft/core";

export default defineDemo({
  id: "basic-walkthrough",
  title: "Basic walkthrough",
  source: {
    baseUrl: "http://localhost:4173",
    initialPath: "/",
  },
  targets: {
    hero: byTestId("hero"),
  },
  async run({ demo }) {
    await demo.scene("landing", async (scene) => {
      await scene.goto("/");
      await scene.establish("hero");
      await scene.caption("Welcome to our product.");
      await scene.hold("3000ms");
    });
  },
});
```

Essa é a demo completa. Um `targets.ts` ou arquivo de configuração separado não é necessário. Você pode extraí-los depois, quando uma demo maior realmente se beneficiar dessa organização.

## O que isso produz

Quando capturada e renderizada, esta demo gera um clipe de aproximadamente 3 segundos:

1. O navegador navega para `/`.
2. A câmera estabelece no elemento `hero` (viewport inteiro).
3. Uma legenda "Welcome to our product." aparece.
4. O estado se mantém por 3 segundos.

<Callout type="info">
O `hold("3000ms")` mais as alocações de frames padrão para `goto`/`establish`/`caption` determinam a duração total. A 60fps, esta demo tem ~180+ frames.
</Callout>

## Rodando

```bash
# inicie seu app em :4173, depois:
pnpm exec democraft render ./demo.ts -o out.mp4
```

Ou itere no Studio:

```bash
pnpm exec democraft studio ./demo.ts
```
