## Usar o Democraft em um projeto

Depois da primeira publicação no npm, instale a CLI e a API de autoria como dependências de desenvolvimento:

<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tab value="pnpm">
    ```bash
    pnpm add -D @democraft/cli @democraft/core
    pnpm exec playwright install chromium
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install --save-dev @democraft/cli @democraft/core
    npx playwright install chromium
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add --dev @democraft/cli @democraft/core
    yarn playwright install chromium
    ```
  </Tab>
</Tabs>

O Chromium é necessário para a captura. Com o app-alvo rodando, valide ou renderize uma demo diretamente:

```bash
pnpm exec democraft validate demo.ts
pnpm exec democraft render demo.ts -o demo.mp4
```

<Callout type="warn" title="Repositório em pré-release">
Os pacotes npm ainda não estão prontos para publicação. Até a primeira release ser concluída, use a configuração de contribuição abaixo. Em especial, a CLI publicada precisa receber um Studio distribuível para que `democraft studio` funcione fora deste monorepo.
</Callout>

## Contribuir a partir do código-fonte

Pré-requisitos: Node.js 20 ou mais recente e pnpm 9 ou mais recente.

```bash
git clone https://github.com/felipersas/democraft.git
cd democraft
corepack enable
pnpm install
pnpm build
pnpm exec playwright install chromium
```

Verifique o workspace:

```bash
pnpm lint
pnpm typecheck
pnpm test
```

<Callout type="info">
Os pacotes do workspace resolvem a partir de seus diretórios `dist/`. Faça um build após a instalação; depois disso, o Turbo recompila as dependências alteradas nos comandos do workspace.
</Callout>

## Resolução de problemas

<Callout type="warn" title="Pacotes do workspace não encontrados">
Se o desenvolvimento local reportar `Cannot find module '@democraft/...'`, rode `pnpm build` na raiz do repositório.
</Callout>

<Callout type="warn" title="Navegador do Playwright ausente">
Se a captura reportar `Executable doesn't exist`, rode `pnpm exec playwright install chromium`.
</Callout>
